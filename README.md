# 🎓 TFG_Trabajo_Final_Ciclo | Fullstack Cycle Project
This repository contains my final degree project, a comprehensive, modular management system featuring a state-of-the-art React frontend and a robust Node.js RESTful API.

# 📝 Overview
This project represents the culmination of my fullstack development studies. It demonstrates an advanced industry-standard architecture, separating concerns with a dedicated Node.js/Express backend API and a dynamic, reactive user interface built with React. This modularity ensures scalability, maintainability, and a superior user experience.

# ⚙️ Installation & Usage
1. Clone the repository:

Bash
git clone https://github.com/tu-usuario/tfg-ciclo-fullstack.git
2. Setup the API:

Bash
cd BackEnd
npm install
npm start # or node src/index.js
3. Setup the Frontend:

Bash
cd FrontEnd
npm install
npm start # typically `react-scripts start` for React apps
Open your browser to the React dev server (typically http://localhost:3000).
